# Drexel Food Truck Interactive Map
Drexel Food Truck Interactive Map is a map displaying food trucks on drexel campus.
## Description
Using a map that you can zoom in and out on, food trucks are displayed with markers that you can interact with. These markers contain information such as times open, menu items, and prices, to allow users to make informed decisions when considering getting food at food trucks.
